about=FR: FuzzyAttributes est un plugin QGIS permettant de convertir des attributs numériques ou catégoriels en valeurs floues (degrés d’appartenance entre 0 et 1) selon plusieurs types de fonctions d’appartenance : linéaire croissante/décroissante, trapézoïdale, triangulaire, sigmoïde (S et Z) et gaussienne. L’interface graphique facilite la configuration des paramètres pour chaque fonction, affiche un exemple visuel de la transformation, et propose une aide intégrée pour comprendre chaque type de fonction floue. Le plugin permet également l’agrégation souple (fuzzy aggregation) de plusieurs critères à l’aide de méthodes paramétrées, avec vérification de cohérence et messages d’avertissement en cas de configurations contradictoires. EN: FuzzyAttributes is a QGIS plugin that converts numeric or categorical attributes into fuzzy values (membership degrees between 0 and 1) using various membership functions: increasing/decreasing linear, trapezoidal, triangular, sigmoid (S and Z), and Gaussian. The graphical interface makes it easy to configure parameters for each function, displays a visual example of the transformation, and provides built-in help to explain each fuzzy function type. The plugin also supports flexible (fuzzy) aggregation of multiple criteria using parameterized methods, with logical consistency checks and warning messages in case of contradictory configurations.
