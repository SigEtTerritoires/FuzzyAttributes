about=FR: **FuzzyAttributes** est un plugin QGIS permettant d’appliquer des transformations floues aussi bien sur des données vecteur que raster.  
- Les **attributs numériques ou textuels** peuvent être convertis en degrés d’appartenance selon plusieurs fonctions floues : linéaire croissante/décroissante, trapézoïdale, triangulaire, sigmoïde (S et Z), ou gaussienne.
- Le module **Fuzzy Aggregation** réalise l’agrégation floue de deux attributs flous selon une fonction subjective d’agrégation définie.  
- Le module **FuzzyRaster** permet d’appliquer directement une transformation floue sur les valeurs d’un raster.  
- Le module **Raster Aggregation** réalise l’agrégation floue de deux rasters selon une fonction subjective d’agrégation définie.  
- Le module **Classes → Fuzzy** permet de reclasser un raster de classes discrètes en valeurs floues selon une table de correspondance.  
L’interface graphique facilite la configuration, affiche des exemples visuels et propose une aide intégrée.  
EN: **FuzzyAttributes** is a QGIS plugin to apply fuzzy transformations to both vector and raster data.  
- Numeric or categorical attributes can be converted into fuzzy membership degrees using several membership functions.  
- The **Fuzzy Aggregation** module performs fuzzy aggregation of two fuzzy attributes according to a defined subjective aggregation function.  
- The **FuzzyRaster** module applies fuzzy functions to raster pixel values.  
- The **Raster Aggregation** module combines two rasters using a subjective fuzzy aggregation function.  
- The **Classes → Fuzzy** module allows reclassification of raster classes into fuzzy values using a mapping table.  
The plugin includes multilingual support (FR, EN, ES, PT), visual previews, contextual help, and metadata logging.
